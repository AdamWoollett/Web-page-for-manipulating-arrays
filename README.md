This is a web page that I created to try out different methods that can be used to add and remove elements from a blank array.
When a user adds elements to the array, they are displayed at the top of the page.
"Add an element to the end via push:" adds whatever the user types in the box to the end of the array using the push() method.
"Remove last element via pop:" removes the last element in the array using the pop() method.
"Remove first element via shift:" removes the first element in the array using the shift() method.
"Add an element to the start via unshift:" adds whatever the user types in the box to the start of the array using the unshift() method.
"Splice values into the array:" allows a user to both add and remove elements at specififc points in the array using the splice() method. This demonstrates the full range by which an array can be manipulated by a user with one function call.
"Remove specififc array values:" allows a user to delete specific items in the array using the splice() method. This demonstrates that the splice() method can work without needing to specify any elements to be added to the array.
"Clear array after a certain position:" allows a user to use the splice() method to delete multiple elements in the array after a specified position. This demonstrates the simplest usage of the splice() method.
